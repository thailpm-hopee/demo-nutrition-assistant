# Smart Nutrition Assistant

A modern landing page for an AI-driven nutrition assistant.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Development
```bash
npm install
npm run dev
```

## Deployment
This project is ready to be deployed on Vercel. Simply connect your GitHub repository to Vercel and it will automatically detect the Vite build settings.
